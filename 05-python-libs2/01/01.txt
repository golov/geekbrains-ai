1. Максимизируюя precision мы минимизируем количество false positive. Максимизируя recall мы минимизируем количество false negatives.

2. F-мера комбинирует precision и recall. Нужно учитывать и false negative и false negatives.
F-мера использует среднее гармоническое, а не среднее арифметическое между precision и recall, что больше наказывает сильные отклонения.

Например в примере где false positive = 0 а false negative = 1, среднее арифметичское значение будет 0.5. F-мера будет 0, и это более полезный показатель, поскольку модель выдающая false positive = 0 будет бесполезной.
